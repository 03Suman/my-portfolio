Hi, I’m Suman Mondal

I am a Django and Python developer currently working on Purohisab 2.0, a financial accounting system for Urban Local Bodies. My work focuses on building secure, reliable and user-friendly modules for voucher processing, financial reporting, municipality management, dashboards and administrative controls.

I work on complex business features such as receipt, payment, contra, other receipt and journal voucher workflows, cheque handling, account-head selection, role-based permissions, user-to-municipality mapping, municipality-wise report generation and automated database handling when a new municipality is registered.

Along with development, I also handle API integration, SQL Server queries, production debugging, deployment support, Gunicorn/Nginx issues, server errors, session/security controls and modern admin UI improvements.
